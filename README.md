# 📊 Customer Analytics Dashboard

This repository features an interactive customer analytics dashboard created with **Power BI Desktop**. It provides data-driven insights into customer demographics, behaviors, and segments — enabling smarter business decisions.

---

## 🔍 Overview

The dashboard focuses on visualizing key metrics and segmenting customers using RFM (Recency, Frequency, Monetary) analysis. It includes:

- 📌 Total Sales & Customer Overview  
- 👥 Demographics by Gender, Age, and Marital Status  
- 📊 Sales Breakdown by Segment  
- 🧠 Customer Segmentation (RFM)  
- 📄 Detailed Customer Table  

---

## 📂 RFM-Based Segmentation

Customers are segmented into:

- 🏆 Champions  
- 💎 Loyal Customers  
- 🛑 At Risk  
- 🚨 Cannot Lose Them  
- 🌱 Promising  
- 😴 About to Sleep  
- 🌟 Potential Loyalists  
- 🆕 New Customers  
- 💤 Hibernating Customers  
- ❌ Lost Customers  
- ❗ Need Attention
